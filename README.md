# priconnecte

Современный веб-мессенджер с эффектом матового стекла (Glassmorphism), написанный на чистом HTML, CSS и JavaScript. Проект демонстрирует работу с LocalStorage, адаптивный дизайн и имитацию живого общения.

## ✨ Особенности

- 🎨 **Glassmorphism UI**: Современный дизайн с размытием, градиентами и полупрозрачностью.
- 💾 **LocalStorage**: История переписки сохраняется в браузере и не пропадает после обновления страницы.
- 🤖 **Smart Bot**: Имитация собеседника, который отвечает случайными фразами с задержкой.
- 📱 **Fully Responsive**: Адаптация под мобильные устройства с выезжающим меню.
- ⌨️ **UX Features**: Отправка по Enter, автоскролл, индикатор набора текста.
- 🕒 **Timestamps**: Время отправки для каждого сообщения.
