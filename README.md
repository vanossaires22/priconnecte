# 💬 Priconnecte

Современный мессенджер с логотипом priconnecte

## ✨ Функции

### 🔐 Авторизация
- Регистрация новых пользователей
- Вход по email и паролю
- Сохранение сессии

### 💬 Чаты
- Личные сообщения
- Групповые чаты
- Каналы
- Конфиденциальные чаты

### 📱 Сообщения
- Текстовые сообщения
- Голосовые сообщения
- Видео сообщения (кружочки)
- Удаление сообщений
- Редактирование

### 📞 Звонки
- Аудиозвонки
- Видеозвонки

### 👥 Контакты
- Поиск контактов
- Добавление контактов
- Управление контактами

### 🎨 Интерфейс
- 3 темы оформления
- Адаптивный дизайн
- Мобильная версия

## 🚀 Запуск

1. Откройте `index.html` в браузере
2. Зарегистрируйтесь
3. Готово!
